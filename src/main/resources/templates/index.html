Auth Security POC

Two demos sharing one Spring Boot app: JWT refresh-token rotation and the OAuth 2.1 authorization-code flow with PKCE.

Demo 1 — JWT refresh token rotation

REST-only. Try it with curl:

curl -s localhost:8080/api/auth/login \
  -H 'content-type: application/json' \
  -d '{"username":"alice","password":"wonderland"}'

Then POST /api/auth/refresh with the refresh token. The old one is invalidated; reusing it revokes the whole token family.
